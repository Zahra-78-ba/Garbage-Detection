
# ♻️ Garbage Detection using YOLOv8 (Streamlit App)

This project is a **web-based Garbage Detection system** built using **Streamlit** and **YOLOv8 (Ultralytics)**.
It allows users to upload images and automatically detects different types of waste objects using a trained deep learning model.

The system demonstrates how Artificial Intelligence can be used to support **smart waste management and environmental awareness**.


##  Features

* 📤 Upload image (JPG / PNG support)
* 🧠 YOLOv8-based object detection
* 🎯 Real-time garbage detection and classification
* 📊 Detection statistics (objects, categories, confidence)
* ⚡ Fast and accurate predictions
* 🎨 Modern dark-themed UI with custom styling
* 🌍 Environment-friendly awareness interface


##  How It Works

1. User uploads an image
2. Image is converted into a format suitable for the model
3. YOLOv8 model processes the image
4. Objects are detected with bounding boxes
5. Results are displayed with:

   * Annotated image
   * Detected object names
   * Confidence scores
   * Summary statistics

---

## 📊 Output Includes

* Detected garbage objects
* Number of total objects
* Number of categories
* Average confidence score
* Visual bounding box results
